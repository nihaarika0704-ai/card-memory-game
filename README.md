# 🧠 Card Memory Game

A fun and interactive Memory Card Game built using React.js.

## 🎮 Features

- Multiple Difficulty Levels (Easy, Medium, Hard)
- Timer Tracking
- Best Time Storage using LocalStorage
- Random Card Shuffle
- Card Matching Logic
- Restart Game Option
- Play Again Feature
- Responsive Design
- Modern Glassmorphism UI

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS3
- Vite
- LocalStorage

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/nihaarika0704-ai/card-memory-game.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
